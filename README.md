# Caliko

The Personal Blog Project Consits of a Django and Django Rest Framework Backend.
The Front End is made using ReactJs.

Backend:
Has 5 models namely, Articles, Blogs, Categories, Payments and Userdetails
The Queries are optimised for speed. 
Appropriate API Views were created for necessary CRUD operations.
Razorpay integration was used for the purpose donation and all API KEYS, SECRET KEYS WERE MADE UNAVAILABLE with the help of .env file

Frontend
The Front End Consists of a State Management system, appropriate cancel tokens for optimisation, a payment gateway integration, lazy-loading and emailJS integration.
The Css was tested for different resolutions.
